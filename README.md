# OceanWatch – Endangered Species Web App

OceanWatch is a full-stack web application focused on endangered marine species. It was built as part of a group university project, with my main contribution centred on the **Endangered Species** feature across both the frontend and backend.

## Tech Stack

- **Frontend:** React
- **Backend:** Java Spring Boot
- **Database:** MongoDB Atlas
- **Testing:** Thunder Client
- **Version Control:** Git / GitHub

## My Contribution

For the Endangered Species section, I:

- built the frontend interface to display species information clearly
- implemented search and filtering so users could explore species in different ways
- developed backend API endpoints in Spring Boot to retrieve and store species data
- connected the frontend to the backend through API calls
- worked with MongoDB Atlas to manage live species data
- tested API endpoints to make sure data was returned and saved correctly

## Features

- View endangered marine species information
- Display species images and key details
- Search species by name
- Filter species in different ways
- Fetch live data from the backend with no hardcoded frontend data
- Add new species data through backend API requests
